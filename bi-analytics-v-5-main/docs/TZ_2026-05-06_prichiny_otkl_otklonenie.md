**Эталон:** `web/AI/msp_*.csv` (Ковенанты `Разрешение на строительство`, `ЗОС`, `СМР`).

### Требования (скрин 5 нижний + скрин 6)

#### Верхний блок «Начало работ» → «Разрешение на строительство»
- [ ] Заменить заголовок «Начало работ» → **«Разрешение на строительство»**.
- [ ] Задача `Разрешение на строительство` принадлежит блоку «Ковенанты».
- [ ] План = `Базовое окончание` задачи.
- [ ] Факт = `Окончание` задачи.

#### Верхний блок «Окончание работ» → «ЗОС»
- [ ] Заменить заголовок «Окончание работ» → **«ЗОС»**.
- [ ] Задача `ЗОС` принадлежит блоку «Ковенанты».
- [ ] План = `Базовое окончание` задачи `ЗОС`.
- [ ] Факт = `Окончание` задачи `ЗОС`.
- [ ] Отображать «отставание дни» = `Базовое окончание − Окончание` по задаче `ЗОС`.

#### «% завершения» → «% завершения СМР»
- [ ] Заменить «% завершения» → **«% завершения СМР»**.
- [ ] Факт = `% завершения` задачи `СМР`.
- [ ] План = ИИ-расчёт: `% выполнения`, который должен быть на **сегодняшний день** исходя из `Базового начала` и `Базового окончания`.
  - Формула: `(today − base_start) / (base_end − base_start) * 100`, ограничено `[0; 100]`.
  - Пример из ТЗ (скрин 6): Базовое Начало `11/1/2025`, Базовое Окончание `5/30/2026`, Дата отчёта `5/6/2026` → План = **89%**.

- [ ] `% план` — расчёт относительно `Базового начала` и `Базового окончания` на дату отчёта.
- [ ] `% факт` — значение из MSP `% завершения`.
- [ ] `Отклонение`: красный шрифт если `> 0`, зелёный если `≤ 0`.
- [ ] Шапка таблицы коррелируется с фильтрами отчёта.
