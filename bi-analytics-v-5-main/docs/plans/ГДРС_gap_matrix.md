| Требование PDF | Где в коде / данных | Статус |
|----------------|---------------------|--------|
| Отчёт «ГДРС», подзаголовок рабочие/техника | `dashboard_technique_tabs`, `dashboard_gdrs_equipment` | ok |
| Отклонение = План − Факт; %(отклонение) от плана | `dashboard_workforce_movement`, эталонная HTML-таблица, графики подрядчиков | ok (сверить цвета с Excel при спорных местах PDF) |
| Фильтры: проект, контрагент, вид ресурсов (колонка H) | фильтры + `_gdrs_detect_csv_resource_kind_column` | ok |
| Динамика: ось неделя / месяц / год | `st.radio` `*_dyn_grain`, агрегация melt→месяц/год; fallback без недель — год | ok (ручная проверка на своих CSV) |
| KPI план / факт / отклонение сверху | `st.metric` после фильтров | ok |
| Сводная таблица: % = факт/план×100 | колонка `%` в сводной по контрагентам | ok |
| Маски FTP `1c_*` / `lc_*` spravochniki | `web_loader._infer_file_type_by_name` | ok |

| Полная замена набора таблиц стр. 11–14 PDF | частично свёрстано в HTML + блоки ниже | **gap** — сверка по скринам |
| История версий ресурсов для ИИ | метаданные загрузки / архив | **gap** — отдельная задача |
| `dashboard_technique` vs единый вход | меню → `dashboard_workforce_movement`; старый отчёт живёт параллельно | **risk** — deeplink/совместимость |
