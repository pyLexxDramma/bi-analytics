3. Подпись «КОМАНДА ПП» vs «КОМАНДА РП» — проверить финальное наименование в ТЗ заказчика.

## Замечания

- **Предупреждения на экране:** централизованные подсказки выводятся только если для строки «Выборка ДС» не найдены обороты 1С **или** не загружен набор TESSA (`tessa_tasks_data`). Если при проверке всё было загружено и сопоставлено — блок предупреждений пустой, это нормально. Оранжевая подсветка ячеек при % выполнения \< 100% — отдельно от `st.warning`.
- **Добавлять на дашборд ничего не требуется** для закрытия этого раунда; открытые вопросы из ТЗ (две колонки РД / РД Техно, подпись КОМАНДА ПП vs РП, админ-статус проекта) — не про предупреждения, а про следующее уточнение ТЗ/функционала при необходимости.
