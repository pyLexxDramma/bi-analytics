# Утвержденный бюджет

## Поля интерфейса ↔ FTP ↔ маппинг
Источники FTP: `web/*_dannye.json`, дополнительно `web/*_spravochniki.json`

| Поле в интерфейсе | Колонка FTP | Canonical/Admin key |
|---|---|---|
| Утвержденный план | `Сумма` (по сценарию утвержденного плана) | `approved plan amount` |
| Факт | `Сумма` (по фактическому сценарию) | `fact amount` |
| Период | `Период` | `period` |
| Проект | `Проект` | `project name` |
| Сценарий | `Сценарий` | `scenario` |

## Чек-боксы
- `Накопительно`: включает нарастающий итог.
- `Показывать отклонение`: добавляет `Факт - План`.

## Фильтры
- **Проект**, **Период**, **Сценарий**, **Статья** (если вынесена).

## Логика расчета
- Выделяется утвержденный контур сценария.
- Сравнивается с фактом.
- Отклонение считается по периоду и (при включении) накопительно.

## Вывод (что видит пользователь)
- **Графики**: сравнение утвержденного плана и факта.
- **Таблица**: период, план, факт, отклонение.

## Контролы UI (как в коде)
- В этой функции явные `st.*` контролы локально не объявлены (используются общие фильтры/обертки).
