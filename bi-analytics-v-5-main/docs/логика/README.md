# Логика дашбордов

В этой папке логика разделена на отдельные файлы по каждому дашборду.

В каждом файле:
- какие файлы используются;
- какие поля читаются;
- вкладки/переключатели/чекбоксы;
- как считаются ключевые показатели;
- как формируется вывод.
