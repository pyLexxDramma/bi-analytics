# График проекта

## Поля интерфейса ↔ FTP ↔ маппинг
Источник FTP: `web/AI/msp_*_<date>.csv`

| Поле в интерфейсе | Колонка FTP | Canonical/Admin key |
|---|---|---|
| Задача | `Название` | `task name` |
| Проект | `ID_проекта` | `project id` |
| План начало | `Начало` | `plan start` |
| План окончание | `Окончание` | `plan end` |
| База начало | `Базовое_начало` | `base start` |
| База окончание | `Базовое_окончание` | `base end` |
| Уровень | `Уровень_структуры` | `level structure` |

## Чек-боксы
- Чек-боксов в типовом режиме нет.
- Если включены служебные опции, они скрывают завершенные/служебные строки.

## Фильтры
- **Проект**: выбирает один проект для timeline.
- **Блок/лот/раздел**: ограничивает задачи внутри проекта.
- **Период**: срез по датам отображения.

## Логика расчета
- Строится таймлайн задач на основе `plan start/plan end`.
- Слой базового плана добавляется из `base start/base end`.
- Для задач считаются отклонения начала/окончания (используются в подсказках и таблице).

## Вывод (что видит пользователь)
- **График**: Gantt/timeline по задачам.
- **Таблица**: список задач с датами плана/базы и отклонениями.

## Контролы UI (как в коде)
- `selectbox`: `Проект (ур. 1)`
- `selectbox`: `Функциональный блок (ур. 2)`
- `selectbox`: `Строение`
