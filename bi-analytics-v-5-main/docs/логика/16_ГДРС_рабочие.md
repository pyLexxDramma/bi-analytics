# ГДРС (рабочие)

## Поля интерфейса ↔ FTP ↔ маппинг
Источники FTP:
- факт: `web/AI/other_*_resursi.csv`
- план: `web/1c_*_spravochniki.json`

| Поле в интерфейсе | Колонка FTP | Canonical/Admin key |
|---|---|---|
| Проект | `Проект` / `Наименование_Проекта` | `project name` |
| Подрядчик | `Подрядчик` / `Наименование_Контрагента` | `contractor` |
| Тип ресурса | `тип ресурсов` | `resource_type` |
| Факт рабочие | дневные/недельные колонки CSV | `fact workers` |
| План рабочие | `КоличествоРаботников` | `gdrs_plan_workers` |
| Источник строки | `data_source` (если есть) | `data_source` |

## Чек-боксы
- `Только рабочие`: исключает строки техники/прочих ресурсов.
- `Показывать долю подрядчика`: добавляет расчет `% от общего факта`.

## Фильтры
- **Проект**, **Подрядчик**, **Период**.

## Логика расчета
- Факт агрегируется из дневных/недельных колонок.
- План подтягивается из `spravochniki.json` по нормализованной связке контрагент+проект.
- `Отклонение = Факт - План`.
- `% выполнения плана = Факт / План * 100` (если `План > 0`).

## Вывод (что видит пользователь)
- **Графики**: план/факт и доля подрядчиков.
- **Таблица**: по подрядчикам/периодам, колонки факт/план/отклонение/%.

## Контролы UI (как в коде)
- В этой функции явные `st.*` контролы локально не объявлены (используются общие фильтры/обертки).
