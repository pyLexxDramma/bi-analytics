# Бюджет план/факт

## Поля интерфейса ↔ FTP ↔ маппинг
Источники FTP:
- `web/AI/msp_*_<date>.csv` (если есть бюджетные поля),
- fallback `web/*_dannye.json`.

| Поле в интерфейсе | Колонка FTP | Canonical/Admin key |
|---|---|---|
| Бюджет план | `Бюджет План` / `Бюджет план` | `budget plan` |
| Бюджет факт | `Бюджет Факт` / `Бюджет факт` | `budget fact` |
| Проект | `ID_проекта` / `Проект` | `project id` / `project name` |
| Лот/раздел | `ЛОТ` / иерархия | `lot` / `section` |
| Период | `Период` (из 1C fallback) | `period` |

## Чек-боксы
- `Использовать fallback 1C`: подставляет план из 1C при пустом плане MSP.
- `Показывать отклонение`: включает столбец/серию `Факт - План`.

## Фильтры
- **Проект**, **Период**, **Лот/раздел**, **тип среза**.

## Логика расчета
- `Отклонение = budget fact - budget plan`.
- Если `budget plan` пуст в MSP, берется из `dannye.json` по ключам проекта/периода.

## Вывод (что видит пользователь)
- **Графики**: план vs факт + отклонение.
- **Таблица**: суммы по проектам/лотам/периодам, экспорт.

## Контролы UI (как в коде)
- В этой функции явные `st.*` контролы локально не объявлены (используются общие фильтры/обертки).
