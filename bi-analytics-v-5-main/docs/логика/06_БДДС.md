# БДДС

## Поля интерфейса ↔ FTP ↔ маппинг
Источник FTP: `web/*_dannye.json`

| Поле в интерфейсе | Колонка FTP | Canonical/Admin key |
|---|---|---|
| Период | `Период` | `period` |
| Сценарий | `Сценарий` | `scenario` |
| Проект | `Проект` | `project name` |
| Статья | `СтатьяОборотов` | `turnover item` |
| Сумма | `Сумма` | `amount` |
| Доход/Расход | `РасходДоход` | `income_expense_flag` |
| Тип статьи | `ТипСтатьи` | `item_type` |

## Чек-боксы
- `Накопительно`: переключает формулу с помесячной суммы на cumulative.
- Дополнительные чек-боксы (если включены) управляют только отображением статей.

## Фильтры
- **Проект**, **Период**, **Сценарий**, **Статья/тип статьи**.

## Логика расчета
- По месяцам: `sum(amount)` по выбранным разрезам.
- Накопительно: `cumsum(sum(amount))` в хронологическом порядке периода.

## Вывод (что видит пользователь)
- **Графики**: bar/line по месяцам или накопительно.
- **Таблица**: свод по проекту, статьям и суммам с экспортом.

## Контролы UI (как в коде)
- `tabs`: `По периодам`
