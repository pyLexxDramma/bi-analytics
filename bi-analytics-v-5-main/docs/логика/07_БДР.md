# БДР

## Поля интерфейса ↔ FTP ↔ маппинг
Источник FTP: `web/*_dannye.json`

| Поле в интерфейсе | Колонка FTP | Canonical/Admin key |
|---|---|---|
| Период | `Период` | `period` |
| Проект | `Проект` | `project name` |
| Сценарий | `Сценарий` | `scenario` |
| Статья БДР | `СтатьяОборотов` | `turnover item` |
| Сумма | `Сумма` | `amount` |
| Доход/Расход | `РасходДоход` | `income_expense_flag` |

## Чек-боксы
- `Накопительно`: переводит график и таблицу в cumulative режим.
- Доп. чек-боксы (если есть) скрывают нулевые статьи.

## Фильтры
- **Проект**, **Период**, **Сценарий**, **Статья**.

## Логика расчета
- Базово: агрегирование `sum(amount)` по статьям/периодам.
- Накопительно: сумма нарастающим итогом по периодам.

## Вывод (что видит пользователь)
- **Графики**: столбчатые (помесячно), линейные/столбчатые (накопительно).
- **Таблица**: статьи и суммы по выбранным фильтрам.

## Контролы UI (как в коде)
- В этой функции явные `st.*` контролы локально не объявлены (используются общие фильтры/обертки).
